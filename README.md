# 🐾 TinDog

A modern and responsive landing page for dog lovers.

TinDog helps dogs find new friends, playmates, and companions nearby through a clean, engaging, and mobile-friendly user experience.

## 🚀 Features

- Responsive design using Bootstrap 5
- Modern hero section
- Testimonials section
- Pricing plans
- Brand logos and social proof
- Mobile-friendly layout
- Animated gradient background

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap 5
